# Register Login

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.2.6.

## Development server

Firstly you should install npm ' npm install' 
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

# Register Form 
- Reactive form yapısını ve validatörlerini kullandım.
İsim ve Soyisim için: required, minlength, maxlength, regex pattern validasyonlarını kullandım.
Tel İçin: required, minlength ve regex pattern validasyonlarını kullandım. 
Email için: required, email validasyonlarını kullandım.
Şifre için: required, minlength ve maxlength validasyonlarını kullandım.
Şifre Tekrarı için: required, mustmach validasyonlarını kullandım.
- Tüm alanlar doğru doldurulduğuda, alanlar kırmızıdan maviye dönecektir. 
- "Üyelik koşulları ve kişisel verilerin korunmasını kabul ediyorum" işaretlendikten sonra register butonu aktif olacaktır.
